📧 שליחת מייל ישירה (EmailJS)

אופציונלי
מה זה? EmailJS מאפשר לשלוח מיילים ללקוחות ישירות מהמערכת בלי לפתוח את ה-Gmail או Outlook. המייל יישלח מהכתובת שלך (balasistore5@gmail.com).

איך להגדיר (פעם אחת בלבד, ~3 דקות):
  1. לך ל-dashboard.emailjs.com/sign-up והירשם בחינם
  2. ב-"Email Services" → "Add New Service" → בחר Gmail וחבר את balasistore5@gmail.com
  3. העתק את ה-Service ID שמופיע (משהו כמו service_abc1234) ↓
  4. ב-"Email Templates" → "Create New Template". הגדר:
    • To Email: {{to_email}}
    • From Name: {{from_name}}
    • Subject: {{subject}}
    • Content: {{{message_html}}} (שלושה סוגריים מסולסלים!)
    העתק את ה-Template ID (משהו כמו template_xyz5678) ↓
  5. ב-"Account" → "General" → העתק את ה-Public Key ↓
  6. הדבק את שלושת הערכים פה ולחץ "שמור".
⚠ מגבלה: EmailJS מאפשר 200 מיילים בחודש בחינם. הקובץ PDF לא נשלח כצירוף — במקום זאת, גוף המייל כולל את כל פירוט החשבונית בפורמט מעוצב. ה-PDF נשמר אצלך במחשב.
